source

de_emperical_co_pc

 de_emperical_co_pc (rslc:str, is_shp:str, idx:str, coh:str, coh_ave:str,
                     az_chunk_size:int=None, n_pc_chunk:int=None,
                     pc_chunk_size:int=None, log=None)

estimate emperical coherence matrix on point cloud data. Only one of n_pc_chunk and pc_chunk_size needs to be setted. The other one is automatically determined. If all of them are not setted, the n_pc_chunk will be setted as the number of azimuth chunks.

Type Default Details
rslc str input: rslc stack
is_shp str input: bool array indicating the SHPs of pc
idx str input: bool array indicating pc
coh str output: complex coherence matrix for pc
coh_ave str output: average value of coherence matrix magnitude
az_chunk_size int None azimuth chunk size, optional. Default: the azimuth chunk size in rslc stack
n_pc_chunk int None number of point chunk, optional.
pc_chunk_size int None chunk size of output zarr dataset, optional. Default: same as is_shp
log NoneType None log file. Default: no log file
